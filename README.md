# Ejercicios de Modelado de Datos con Entity Framework Core :clipboard:

En estos ejercicios, se exploran conceptos de modelado de datos utilizando Entity Framework Core y relaciones entre entidades en una base de datos MySQL. A continuación, se presenta un resumen de cada ejercicio:

## :one: Ejercicio 1: Modelado de Artículos y Fabricantes

Este ejercicio involucra el modelado de entidades "Articulo" y "Fabricante" en una base de datos. Se establece una relación uno a muchos entre un fabricante y sus artículos.

**Archivos involucrados:**
- `AppDbContext.cs`
- `Articulo.cs`
- `Fabricante.cs`

## :two: Ejercicio 2: Modelado de Empleados y Departamentos

En este ejercicio, se definen las entidades "Empleado" y "Departamento", y se establece una relación uno a muchos entre un departamento y sus empleados.

**Archivos involucrados:**
- `AppDbContext.cs`
- `Empleado.cs`
- `Departamento.cs`

## :three: Ejercicio 3: Modelado de Almacenes y Cajas

El tercer ejercicio abarca las entidades "Almacen" y "Caja", y se establece una relación uno a muchos entre un almacén y sus cajas.

**Archivos involucrados:**

- `Almacen.cs`
- `Caja.cs`

## :four: Ejercicio 4: Modelado de Salas y Películas

En este último ejercicio, se modelan las entidades "Sala" y "Pelicula", y se establece una relación uno a muchos entre una película y sus salas de proyección.

**Archivos involucrados:**
- `AppDbContext.cs`
